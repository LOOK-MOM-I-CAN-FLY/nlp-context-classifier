# nlp-context-classifier

Этот проект представляет собой NLP-систему для бинарной классификации текста. Основная задача модели — определять, содержит ли текст политизированный контекст, связанный с военными конфликтами, Россией, Украиной и тематикой СВО, либо упоминания являются нейтральными и относятся, например, к истории, географии, культуре, языкам или другим неполитическим темам.
