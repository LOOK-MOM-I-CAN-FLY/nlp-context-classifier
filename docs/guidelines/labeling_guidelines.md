Класс 1 (политизированный/военный контекст):
- Упоминание военных действий, обстрелов, наступлений, потерь
- Санкции, дипломатические разрывы, высылка послов
- Военная помощь, поставки оружия
- Военные лидеры, командиры, батальоны
- Беженцы в контексте войны
- Оккупация, освобождение территорий
- Риторика про «Запад vs Россия», «агрессия», «вторжение»

Класс 0 (нейтральный контекст):
- Географические упоминания без конфликтного контекста
- Культура, история до 2014 года (если нет отсылки к войне)
- Спорт, еда, туризм, язык
- Научные публикации, академические тексты
- Чисто экономические новости без военной привязки

Сложные случаи:
- Экономика + санкции → класс 1
- История Крыма без привязки к 2014+ → класс 0
- Упоминание «войны» в историческом контексте (ВОВ) → класс 0
